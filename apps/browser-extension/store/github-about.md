## Description courte

StreamVolume Guard Hub protège les lives des pics audio et stabilise le volume entre YouTube, Twitch, TikTok, Kick et autres sites vidéo. Open source, sans tracker, sans collecte de données : aucune donnée récupérée, traitement local.
