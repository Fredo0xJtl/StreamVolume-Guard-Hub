# Plans Historiques

Ce dossier contient des plans de travail produits pendant les etapes de construction.

Ils ne sont pas la source de verite actuelle du projet hybride. Certains fichiers conservent volontairement des chemins historiques comme `D:\Codex\App StreamVolume Guard` ou l'ancien dossier `desktop/` parce qu'ils documentent comment le prototype a ete construit.

Pour travailler aujourd'hui, utiliser plutot :

- `README.md`
- `AGENTS.md`
- `docs/hybrid-architecture.md`
- `docs/product-next-plan.md`
- `docs/tester-checklist.md`
- `packages/protocol/README.md`
- `apps/desktop/README.md`
- `apps/browser-extension/README.md`

Ne pas copier les commandes anciennes de ces plans sans les adapter au dossier actuel `D:\Codex\StreamVolume Guard Hybride` et au chemin `apps/desktop/`.
